# Site iAgentes

Site institucional da iAgentes - Treinamento profissional de agentes de IA.

## 🎯 Proposta de Valor

Treinamos agentes de IA sob medida para empresas que querem automação inteligente sem perder qualidade.

## 🚀 Tech Stack

- **Framework:** Vite + React + TypeScript
- **UI:** shadcn/ui + Tailwind CSS
- **Animações:** Framer Motion
- **Forms:** React Hook Form + Zod
- **Backend:** Supabase (forms + analytics)

## 📦 Instalação

```bash
npm install
npm run dev
```

## 🏗️ Estrutura

- `src/components/` - Componentes React
- `src/pages/` - Páginas (landing page única)
- `src/config/` - Configurações e endpoints
- `public/` - Assets estáticos

## 🎨 Design System

Baseado no site iAvançada, adaptado para o contexto de treinamento de agentes de IA.

### Cores
- Primary: Vermelho vibrante (identidade iAgentes)
- Background: Tema claro/escuro
- Gradientes: Primary com transparência

### Componentes
- Hero com animação 3D
- Trust bar com métricas
- 3 pilares da oferta
- Cases de sucesso
- Timeline do processo
- ICP (quem é ideal)
- FAQ
- Form de contato

## 📝 Deploy

```bash
npm run build
```

Deploy via Vercel/Netlify ou qualquer plataforma que suporte SPA.
